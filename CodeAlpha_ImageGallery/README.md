# CodeAlpha Image Gallery

A small responsive image gallery demo (HTML/CSS/JS) you can host directly from GitHub Pages or open locally.

Features
- Responsive CSS Grid gallery
- Hover effects and smooth transitions
- Lightbox with next/prev and keyboard navigation
- Category filters (All / Nature / City / Animals)

How to view
1. Open `index.html` in a browser (double-click) or use a simple static server such as Live Server in VS Code or `python -m http.server`.

Files
- `index.html` – main page
- `css/gallery.css` – styles
- `js/gallery.js` – client-side behavior
- `images/` – image assets
